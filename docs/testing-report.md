## 5. Тестовая инфраструктура

**Бэкенд (`conftest.py`):**
- Изолированная SQLite-БД создаётся перед каждым тестом и удаляется после
- Фикстура `client` — FastAPI TestClient с подменённой зависимостью `get_db`
- Фикстура `registered_user` — зарегистрированный студент `student@uni.ru`
- Фикстура `auth_headers` — заголовок `Authorization: Bearer <token>` для студента
- Фикстура `staff_headers` — заголовок `Authorization: Bearer <token>` для сотрудника

**Фронтенд:**
- `test-setup.ts` подключает `@testing-library/jest-dom`
- Zustand-сторы сбрасываются через `setState({...})` в `beforeEach` перед каждым тестом
